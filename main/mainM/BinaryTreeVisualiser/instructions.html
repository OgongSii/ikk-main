Instructions

여기에서 이진트리 시각와의 사용법을 숙지하실 수 있습니다.
사용법은 두 가지로 나뉩니다 여기에서는 바이너리 힙 + 우선순위 큐 (랜덤 바이너리 힙, 바이너리 힙 빌드, 삽입, 삭제, 최대 추출, 최대 가져오기, 힙 정렬)을 사용 할 수 있습니다. 여기에서는 이진 탐색 트리 (이진 탐색 트리, 삽입, 검색, 삭제, 최대값 가져오기, 최소값 가져옿기, 선행자 가져오기, 후임자 가져오기, to preorder array, to inorder (sorted) array, to postorder array).
이진 탐색 트리, 이진 힙 또는 나열된 알고리즘이 무엇을 의미하는지 잘 모르겠으면 여기 를 방문하세요.
표기를 이해하기 힘드시면 Icon 를 참고하세요. 표기릐 뜻을 알고싶으시면 Control 를 참고하세요.

Controls

Animation controls

아래 버튼은 현재 재생 중인 애니메이션을 제어하며 이전 애니메이션이나 다음 애니메이션으로 이동할 수 있습니다. 한 번에 재생 버튼만 표시되거나 일시 중지 버튼만 표시됩니다. 적절한 명령이 없으면 버튼이 비활성화됩니다.

Button Meaning Description
애니메이션 재생 이 버튼은 일시 중지된 애니메이션을 재생하기 시작합니다. ( 자동으로 애니메이션이 재생됩니다. )
애니메이션 중지 이 버튼은 애니메이션 재생을 일시 중지합니다.
애니메이션을 다시 재생 애니메이션을 다시 재생합니다.
애니메이션 건너뛰기 어떤 애니메이션은 시간이 오래 걸립니다. 이 버튼을 누르면 애니메이션을 건너뛸 수 있습니다. 애니메이션이 스킵되어 다음 알고리즘(있는 경우)을 재생하지 않습니다.
이전 애니메이션 이전 알고리즘(애니메이션) 으로 이동하여 재생을 시작합니다.
다음 애니메이션 다음 알고리즘(애니메이션) 으로 이동하여 재생을 시작합니다.
애니메이션을 차례로 재생 이 체크박스를 선택하면 현재 애니메이션이 종료된 후 다음 애니메이션이 자동으로 시작됩니다.
Speed of move:
이동 속도 설정 애니메이션에는 두 가지 단계가 있습니다. 첫 번째는 그래픽 요소를 이동하는것 두 번째는 다른 무언가를 보여주는 데 시간이 걸립니다. 이 버튼은 첫 번째 유형의 단계를 설정합니다.
Duration of a step:
단계 기간 설정 위 설명과 같이, 이 버튼은 다른 종류의 애니메이션 지속 시간을 설정합니다.

History controls

실행된 모든 알고리즘은 기록에 표시됩니다. 다시 재생하거나 기록를 변경할 수 있습니다.

Button Meaning Description
실행된 알고리즘 목록 실행된 모든 알고리즘이 나열됩니다. 첫 번째는 목록의 맨 위에 있습니다. 새로운 알고리즘이 끝에 추가됩니다. 이전 알고리즘(애니메이션)을 재생한 다음 다른 알고리즘을 실행하여 기록을 변경할 수 있습니다. 현재 알고리즘(애니메이션)이 선택되었습니다. 두 번 클릭하여 다른 항목을 선택할 수 있습니다.

Algorithms controls

아래의 버튼들을 클릭하면 선택한 알고리즘을 실행하고 애니메이션을 보여줍니다. 또한 알고리즘에 추가합니다 history, 그리고 애니메이션을 다시 재생할 수 있습니다.
일부 알고리즘은 입력이 필요합니다. 노드 또는 값입니다. 텍스트 입력 시 문자열로 값이 입력됩니다. 노드를 클릭하여 선택합니다.
일부 알고리즘은 값을 반환합니다. 노드인 경우 애니메이션이 끝날 때 노드가 선택됩니다. 배열인 경우, 배열은 애니메이션의 끝에 표시됩니다.
이 버튼은 애니메이션이 중지된 경우에만 활성화됩니다 - 마지막 애니메이션이 모두 재생되었습니다.
이러한 알고리즘에 대한 자세한 정보를 알고 싶다면, 여기에서 참고하세요 pseudocodes.

Algorithm (Binary Heap) Meaning Description
min: max: 무작위 힙 생성 랜덤 이진 힙을 만듭니다. 생성된 노드의 값은 지정된 간격에서 가져옵니다.
array: 힙 생성 지정된 배열에서 이진 힙을 빌드합니다. 배열은 "," (쉼표)로 구분된 값의 문자열로 입력됩니다.
value: 노드 삽입 지정된 값을 사용하여 새 노드를 생성하고 힙에 삽입합니다.
selected node 노드 삭제 선택한 노드를 힙에서 삭제합니다.
노드의 최대값 추출 루트 노드 삭제와 동일합니다. 최대값을 가진 노드는 항상 이진 힙의 루트입니다.
최대값 노드 가져오기 트리의 루트를 반환합니다. 하지만 최대값을 가진 노드는 항상 이진 힙의 루트입니다.
힙 정렬 이 알고리즘은 힙을 정렬합니다. 결과는 배열로 표시됩니다.
Algorithm (Binary Search Tree) Meaning Description
min: max: 무작위 검색 트리 생성 무작위 이진 검색 트리를 만듭니다. 생성된 노드의 값은 지정된 간격에서 가져옵니다.
value: 노드 삽입 지정된 값을 사용하여 새 노드를 만들고 이 노드를 이진 검색 트리에 삽입합니다.
value: 탐색 노드 지정된 값을 가진 노드의 첫 번째 항목을 찾습니다.
selected node 노드 삭제 이진 탐색 트리에서 선택한 노드를 삭제합니다.
of selected (sub)tree 노드의 최대값 선택한 하위 트리의 최대값을 가진 노드를 찾아 반환합니다. 루트를 선택하여 하위 트리를 선택합니다. 노드를 선택하지 않으면 전체 트리의 루트가 선택됩니다.
of selected (sub)tree 노드의 최소값 선택한 하위 트리의 최소값이 있는 노드를 찾아 반환합니다. 루트를 선택하여 하위 트리를 선택합니다. 노드를 선택하지 않으면 전체 트리의 루트가 선택됩니다.
of selected node 전임자 찾기 선택한 노드의 이전 노드를 가져옵니다. 이전 노드 값이 처음보다 낮은 노드입니다.
of selected node 후임자 찾기 선택한 노드의 후속 노드를 가져옵니다. 첫 번째 다음 값이 더 높은 노드입니다.
노드를 전위 배열로 정렬합니다. 전위로 트리를 이동합니다. 전위 배열을 표시합니다.
(To Sorted Array) 노드를 순서대로(정렬된) 배열로 정렬합니다. 순서대로 트리를 정리합니다. 순서 배열을 표시합니다. 이 배열은 값별로 정렬됩니다.
노드를 후위 배열로 정렬합니다. 후위로 트리를 이동합니다. 후위 배열을 표시합니다.
